# Audit Express de Site Web

## Description
Landing page pour proposer un service d'audit express de site web aux indépendants francophones qui ont un site ancien ou inefficace.

## Cible
Indépendants francophones (consultants, artisans, professions libérales, etc.) qui possèdent un site web mais qui ne génère pas suffisamment de clients ou de prospects.

## Promesse cash-first
Service d'audit express de site web à 97€ avec une réduction de 20% pour les premiers clients (early bird). L'audit permet d'identifier rapidement les problèmes techniques et marketing qui empêchent le site de convertir efficacement les visiteurs en clients.

## Fonctionnalités
- Outil interactif d'analyse préliminaire de site web
- Formulaire de capture de leads pour réserver un audit complet
- Présentation des bénéfices concrets pour les indépendants
- Tarification claire avec offre early bird

## Type de CTA
- **Lead**: Capture d'email et d'informations de contact pour proposer le service d'audit

## Comment lancer en local
1. Cloner ce dépôt
2. Ouvrir le fichier `index.html` dans un navigateur web
3. Pour tester la fonctionnalité de capture de leads, vous devez configurer une API compatible avec le format attendu par le script

## Technologies utilisées
- HTML5
- CSS3
- JavaScript vanilla

## Variables d'environnement
Pour que la capture de leads fonctionne correctement, configurez la variable `KOMPOS_LEADS_API_ORIGIN` dans votre environnement avec l'URL de votre API de leads.
